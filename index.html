Language web utilisées pour créer et concevoir les apparences de vos sites Web. HTML ajoute la structure et CSS le style .

JavaScript

JavaScript est un langage de programmation qui vous permet d'animer les pages web, il s'agit par exemple de cartes interactives ou de graphiques animés en 2D/3D.

NodeJS

Bien qu'à l'origine JavaScript ait été conçu pour fonctionner sur les navigateurs, il est maintenant possible d'exécuter des applications écrites dans ce langage sur des serveurs web grâce à Node.js.

Git est l'outil par excellence que tout programmeur doit maîtriser. Git est un logiciel de gestion de versions décentralisé.
